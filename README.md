# Model Context Protocol Example

This Python example demonstrates the Model Context Protocol (MCP) concept. It shows how a single 'Product' model can be interpreted and presented differently based on its operational 'context' (e.g., catalog display vs. order processing).

## Language

`python`

## How to Run

Save the code as `mcp_example.py` and run it from your terminal using: `python mcp_example.py`
